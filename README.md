## 🛡️ أساسيات السيقورتي

### 🔑 مصطلحات هامة ستستمر معنا طوال مسيرتنا:

* **Authentication 🔐 (المصادقة):** إثبات ملكية الحساب (تأكيد هويتك)، ولها ثلاثة عوامل رئيسية:
  1. *شيء تعرفه:* مثل كلمة المرور أو الرمز السري.
  2. *شيء تملكه:* مثل بطاقة الهوية الذكية (Smart Card).
  3. *شيء فيك (القياسات الحيوية):* مثل بصمة العين أو بصمة الإصبع.

* **أنواع المصادقة:**
  * **SFA (Single-Factor Authentication):** مصادقة عامل واحد.
  * **MFA (Multi-Factor Authentication):** مصادقة متعددة العوامل.

* **Authorization 😎 (الصلاحيات):** ما يُسمح لك برؤيته أو تعديله من بيانات بعد إتمام المصادقة.
